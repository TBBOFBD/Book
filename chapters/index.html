Introduction

What Is This Book ?

This is a book about modding and reverse engineering video games.
It is a collection of tutorials, guides, and other resources.

This book is going to be told in a story format.
The story will be told in a series of chapters, each chapter will be about a different topic.

It is a work in progress, and is currently in the early stages of development.
If you would like to contribute, please visit the GitHub repository!

What Is Modding ?

Modding is the process of modifying a game to change its content, gameplay, or other aspects.

Is this book for me ?

If you are interested in modding video games, then this book is for you!
This book is intended for beginners, and assumes no prior knowledge of modding.
If you are an experienced modder, you may find this book useful as a reference.
If you are a game developer, you may find this book useful for learning about posible vulnerabilities in your games.
